# Repositório de Projetos C

Este repositório contém projetos desenvolvidos durante os estudos de programação, utilizando a linguagem C++. Os códigos foram criados com foco em prática acadêmica, desenvolvimento da lógica de programação e aplicação de conceitos fundamentais da computação.

## Estrutura do Repositório

O repositório está organizado em diretórios separados por tema ou finalidade dos projetos.

### Projetos Faculdade

Projetos desenvolvidos como atividades acadêmicas e exercícios práticos.

#### Cálculo de Média
Programa simples responsável pelo cálculo de médias numéricas utilizando estruturas básicas da linguagem C.

#### Leitura de Matrizes
Projeto voltado para manipulação e leitura de matrizes, utilizando estruturas bidimensionais e operações fundamentais.

#### Sistema de Biblioteca
Sistema simples para gerenciamento de biblioteca, desenvolvido em linguagem C com foco em lógica de programação e organização de dados.

### Projetos Pessoais

Projetos desenvolvidos para prática individual e aprimoramento das habilidades em programação.

#### AUTH_Clientes
Sistema simples de autenticação de clientes desenvolvido em C++, utilizando apenas recursos básicos da linguagem.

## Tecnologias Utilizadas

- Linguagem C

## Objetivos do Repositório

- Praticar lógica de programação
- Desenvolver familiaridade com C++
- Organizar projetos acadêmicos e pessoais
- Registrar evolução nos estudos de programação
